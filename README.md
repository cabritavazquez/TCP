Conexión TCP 
Servidor - Cliente a través del puerto 4950 (TCP / UDP: TCP)

Para poner en funcionamiento el TCP tendremos que ejecutar primero el server y después el cliente.

1.    >./server


2.    >./client


3.  Hasta que mandes "fin"
